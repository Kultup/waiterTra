- 🍽️ Ресторани

- 🍺 Паби

## 🍽️ Типи тестів для офіціантів

### 1. Сервірування (Virtual Desk) ✅
**Навички:** Сервірування столу, розміщення приладів

**Приклади завдань:**
- Розмістити ножі, виделки, ложки у правильному порядку
- Розставити келихи для вина, води, шампанського
- Розмістити тарілки (закусочна, основна, десертна)
- Сервірувати стіл для різних типів подачі (сніданок, обід, вечеря, банкет)

**Критерії оцінювання:**
- Правильність розміщення (відстань, порядок)
- Відповідність типу заходу
- Час виконання

---

### 2. Ігрові сценарії (Game Scenarios) ✅
**Навички:** Комунікація з гостями, вирішення конфліктів

**Приклади сценаріїв:**
- "Гість скаржиться на холодну страву"
- "Алергія у відвідувача"
- "Гість поспішає на потяг"
- "Розбите скло на підлозі"
- "Гість замовляє вино — запропонувати декантацію"

**Критерії оцінювання:**
- Правильність вибору відповіді
- Дотримання стандартів сервісу
- Емпатія та професіоналізм

---

### 3. Quiz-тести (Тести з варіантами) ✅
**Навички:** Теоретичні знання

**Приклади тем:**
- **Винна карта:** види вин, температури подачі, келихи
- **Страви:** інгредієнти, способи приготування, подача
- **Напої:** кава, чай, коктейлі, міцний алкоголь
- **Етикет:** правила поведінки, робота з приладами
- **Безпека:** HACCP, гігієна, терміни зберігання

**Приклад питання:**
```
Яка температура подачі білого вина?
○ 6-8°C
○ 8-10°C ✓
○ 12-14°C
○ 16-18°C
```

---

### 4. Комплексні тести (Complex Test)
**Навички:** Комбіновані завдання

**Приклади:**
- Повне обслуговування гостя (від зустрічі до розрахунку)
- Робота з POS-системою
- Підготовка звіту за зміну
- Інвентаризація
